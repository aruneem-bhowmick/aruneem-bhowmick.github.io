⚡ Self-host Space Mono, Roboto, and Roboto Slab

Replace the render-blocking Google Fonts CDN requests with local
@fontsource packages for the React app to eliminate cross-origin
network round trips. During the migration, audit font weight usage
and drop the unused 300 weight for both Roboto and Roboto Slab to
further shrink the payload.

For the static 404 page, manually vendor the required Space Mono
woff/woff2 files (weights 400 normal) into public/fonts/ and declare
them with inline @font-face rules, mirroring the existing Cascadia
Code setup.

This fully removes all dependencies on fonts.googleapis.com and
fonts.gstatic.com across the entire site.
